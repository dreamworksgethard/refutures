# Refutures AI

Solana futures trading agent — main website and DApp.

## Sites

| Site | Path | Description |
|------|------|-------------|
| **Main Website** | `/index.html` | Home, about, how it works, tokenomics, roadmap |
| **DApp** | `/dapp/index.html` | SOL futures agent trading console |

## Quick Start

```bash
python3 -m http.server 8080
```

- Main site: [http://localhost:8080](http://localhost:8080)
- DApp: [http://localhost:8080/dapp/](http://localhost:8080/dapp/)
